## Списки

Чтобы добавить ненумированные списки, необходимо пункты выделить (*). Например, вот так:
* Элемент 1
* Элемент 2

Чтобы добавить нумированные списки, необходимо просто пронумеровать. Например, вот так:
1. Пункт первый
2. Пункт второй
